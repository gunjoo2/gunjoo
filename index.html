학교 작품

선문대학교 컴퓨터공학과를 재학하면서 제작한 프로젝트 결과물

' 다양한 경험 '
IT의 다양한 분야 를 체험해 보고 싶기에 프로젝트 주제 역시 다양한 언어를 사용하여 진행을 하였습니다. '아두이노를 이용한 임베디드 프로그램 개발 ', '오픈소스 공공데이터와 JSP를 이용하여 전국의 공공화장실 위치를 알려주는 웹 지도 개발 ', 'UNITY를 이용한 안드로이드 앱 게임 개발 ', '아두이노와 파이썬 웹 프레임워크 FLASK를 이용한 웹 기반 거짓말 탐지기 개발 '을 하였습니다.

' 마음 굳힌 분야는 웹 개발  '
다양한 프로젝트를 해보았지만 제가 흥미를 느끼고 관심이 많았던 분야는 웹 이었습니다.
다른 프로그램이나 세팅이 필요없이 HTML,CSS,JAVASCRIPT를 서버에 올려두기만 한다면 기기 제약이나 추가 프로그램 설치 필요없이 브라우저만 있다면 결과물을 볼 수 있다는 것에 큰 동기부여가 되었습니다.
